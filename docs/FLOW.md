# 智汇就业 — 功能流程速查

## 用户端（求职者）

```
打开小程序
  │
  ▼
┌─────────────────────┐
│     登录页          │  ← 手机号 + 验证码（演示版自动通过）
│ 输入手机号 → 登录   │
└─────────┬───────────┘
          │
          ▼
┌─────────────────────┐
│     首页            │
│                     │
│  ├─ 推荐岗位        │  ← 热门标记优先，按报名人数排序
│  ├─ 语音找工作      │  ← 🎤 按住说话
│  ├─ 快捷功能        │  ← 找工作/报名/签到/工资
│  └─ 底部导航        │
└─────────┬───────────┘
          │
          ├──→ 岗位列表（分类/排序/搜索）
          ├──→ 岗位详情（报名入口）
          ├──→ 我的报名
          ├──→ 工资
          └──→ 「我的」
                  │
                  ├── 完善资料 ← 身份证号/地址/照片/贫困
                  ├── 进入企业端
                  └── 退出登录
```

## 企业端

```
我的 → 进入企业端
  │
  ├── 发布岗位 → 填写表单 → 上传营业执照 → 提交审核
  │
  └── 岗位管理 → 查看已发布
        ├── 待审核/驳回 → 编辑 → 重新提交
        └── 已通过 → 删除
```

## 管理员端

```
管理后台 → 岗位审核
  │
  ├── 待审核 tab
  │     ├── 查看证明材料（点击放大）
  │     ├── 通过 → 岗位发布
  │     └── 驳回 → 填写原因
  │
  ├── 全部岗位 tab
  │     ├── 标记/取消热门 🔥
  │     └── 撤销发布（审核错了可纠正）
  │
  └── 导出求职者（CSV/Excel）
```

## 数据库一览

```
accounts     用户账号       ← 登录身份
jobs         岗位数据       ← ID: {日期}-{岗位名}-{企业名}
jobseekers   求职者档案     ← ID: {日期}-{姓名}
applications 报名记录
wage_statements 工资单
checkins     签到记录
policies     政策资讯
```

## 云存储路径

```
cert/         ← 企业的营业执照
idcard/       ← 求职者的身份证照片
export/       ← 管理员导出的CSV文件
```
